# Student Management System: CRUD Operations & OOP Design

[Italiano](#italiano) | [English](#english)

---

## <a name="italiano"></a> 🇮🇹 Versione Italiana

### 📝 Descrizione
Un'applicazione Java robusta progettata per la gestione accademica. Implementa il ciclo di vita completo dei dati (CRUD) garantendo un'architettura modulare e pulita.

### 🛠️ Tecnologie
* **Linguaggio:** Java 17+
* **Struttura Dati:** ArrayList (Gestione dinamica)
* **Principi:** OOP (Incapsulamento, Separazione delle responsabilità)

### 🚀 Funzionalità
- **Create**: Aggiunta studenti con ID univoco.
- **Read**: Visualizzazione elenco formattato.
- **Update**: Modifica voti tramite ID.
- **Delete**: Rimozione record dalla memoria.

## <a name="english"></a> 🇬🇧 English Version

### 📝 Description
A robust Java application designed for academic management. It implements the full data lifecycle (CRUD) ensuring a modular and clean architecture.

### 🛠️ Tech Stack
* **Language:** Java 17+
* **Data Structure:** ArrayList (Dynamic management)
* **Principles:** OOP (Encapsulation, Separation of Concerns)

### 🚀 Key Features
- **Create**: Add new students with a unique ID.
- **Read**: Display a formatted student list.
- **Update**: Update grades by searching for a specific ID.
- **Delete**: Remove records from the dynamic collection.
